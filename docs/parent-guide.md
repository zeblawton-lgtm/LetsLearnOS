# Parent guide

## First start

A new database creates two generic sample profiles: `Learner One` (age 5) and
`Learner Two` (age 3). Use the parent area to replace these names and choose
profiles that fit your household. No personal profile data is included in the
repository.

The younger experience uses fewer items, larger visuals, gentler tracing, and
positive-only feedback. The older experience adds more arithmetic, patterns,
word problems, and denser puzzle layouts.

## Learning modules

LetsLearnOS includes math, Spanish, world maps, a 3D globe, space, rocket
science, an interactive science lab, stories, coloring, tracing, connect the
dots, memory, puzzles, stickers, piano, mazes, worksheets, and seek-and-find
activities. Math and worksheet content is deterministic and never generated by
an AI model.

## Narration and audio

Browser SpeechSynthesis provides offline narration. An administrator may enable
OpenAI text-to-speech by placing an API key in the backend `.env`; no key is
entered in the child or parent browser UI.

When OpenAI is enabled, the narration voice is AI-generated. The narrated text
is static learning copy. LetsLearnOS does not send profile names, progress, PINs,
or database records to OpenAI. If the API is unavailable, the app automatically
uses the device voice.

Sound effects are synthesized locally. The public build ships no commercial
music; a properly licensed local asset pack is optional.

## Parent area

Use the lock control and enter the four-digit parent PIN. The parent area can:

- mute narration/sound and optional music;
- change the parent PIN;
- open printable worksheets; and
- end the current session.

Change the bootstrap PIN before normal use. PIN values are never stored in
plaintext.

## Progress and privacy

Profiles, sessions, attempts, settings, and cached narration audio remain in the
local database/cache on the kiosk. Sessions are not time-limited; recorded time
appears only in progress summaries.

## Optional asset pack

The public repository uses neutral fallback artwork. Administrators may add a
local artwork/music pack only when they have the right to use it. See
`README.md` and `docs/credits.md` for locations and licensing boundaries.

## Maintenance

From the parent account, view or restart the service with:

```bash
sudo -u kids XDG_RUNTIME_DIR=/run/user/$(id -u kids) \
  journalctl --user -u letslearnos.service -n 100

sudo -u kids XDG_RUNTIME_DIR=/run/user/$(id -u kids) \
  systemctl --user restart letslearnos.service
```

For upgrades, run `sudo bash scripts/update.sh` from the Git checkout. It
preserves the installed `.env` and SQLite database.
