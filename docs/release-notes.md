# LetsLearnOS release notes

## 1.0.0 — public baseline

- Rebranded and generalized the educational kiosk as LetsLearnOS.
- Began a clean public Git history without private infrastructure or provider
  tooling.
- Added optional server-side OpenAI text-to-speech with disk caching and offline
  browser SpeechSynthesis fallback.
- Added clear AI-generated-voice disclosure and backend-only credential rules.
- Replaced personal first-run profiles with generic learners.
- Removed bundled proprietary character images and commercial music.
- Added public asset, secret, private-host, and provider-reference audits.
- Renamed package, service, install, database, cache, and ISO identifiers.
